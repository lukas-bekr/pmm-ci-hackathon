# PMM Competitive Intelligence Hackathon

A collaborative hackathon project focused on building competitive intelligence tooling for Product Marketing Managers (PMMs).

## About

This repository is the central hub for the PMM CI Hackathon. Contributors will build tools, templates, and workflows to help PMMs gather, analyze, and act on competitive intelligence data more efficiently.

## Project Goals

- **Automate** competitive data collection from public sources
- **Analyze** competitor positioning, pricing, and feature sets
- **Visualize** competitive landscape and trends
- **Enable** data-driven product and marketing decisions
